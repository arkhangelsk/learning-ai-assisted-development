# Claude Common Slash Commands
Claude Code has many built-in slash commands. Here are some of the most commonly used ones:

1. **/help** - Displays a list of available commands and their descriptions.
2. **/model [model_name]** - Switches the active model to the specified one (e.g., `claude-2`, `claude-instant-100k`).
3. **/compact** - Compacts the current context to save memory and improve performance.
4. **/context** - Shows the current context that Claude is using for code generation.
5. **/clear** - Clears the current context and conversation history.
6. **/settings** - Opens the settings menu where you can configure your `settings.json` file.
7. **/verbose** - Toggles verbose mode, which provides more detailed output and explanations for generated code.
8. **/hooks** - Displays the list of configured hooks and their statuses.
9. **/tools** - Lists the tools that Claude has access to and their current permissions.
10. **/memory** - Shows the contents of the current project memory, including any `CLAUDE.md` or `.github/copilot-instructions.md` files.
11. **/run [command]** - Executes a shell command directly from the chat interface (requires permission).
12. **/test** - Runs the test suite for the current project (requires permission).
13. **/lint** - Runs the linter for the current project (requires permission).
14. **/deploy** - Deploys the current project to the configured environment (requires permission).
