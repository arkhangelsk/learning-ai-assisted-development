## AI Essentials for Software Engineers

### Objective: 

As a software engineer, it's important to understand:

- How AI integrates into frontend and backend architectures
- How applications interact with large language models through APIs
- How to use AI to support code reviews, testing, deployment, observability, and incident analysis
- How AI-powered features are designed, implemented, tested, and released
- Common failure modes of AI systems and how they behave in real-world production environments
- How to monitor, troubleshoot, and improve AI systems after deployment

### Things to know:

- AI coding assistants (GitHub Copilot, Claude Code, Codex) - learn at least one AI coding assistant deeply enough to use it effectively
- Understanding LLM Capabilities: Claude, GPT, and Open Source Models — Choosing the Right Model for the Right Task
- LLM APIs: Connecting Applications to Large Language Models
- Understanding prompting techniques
- Managing tokens, context limits, and costs
- RAG, agents, subagents, multi-agent workflows, loop engineering
- MCP/Tooling, Agent Skills
- Security, privacy, and data handling considerations
